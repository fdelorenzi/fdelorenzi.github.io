Vibe-Coding Security: Securing AI-Generated Codebases
The AI coding revolution shipped a security crisis. This talk covers the real-world vulnerabilities emerging from AI-generated codebases pushed to production without security review: misconfigured Supabase RLS, exposed service role keys, SECURITY DEFINER functions in public schema, and more. Live demos of scanning and exploitation.
PugliaTechs, 2025 · Bari, Italy

Think like a Hacker: Protect Web and LLM Applications
How does a hacker think when targeting web applications and Large Language Models? This interactive session walks through vulnerabilities affecting both traditional web apps and LLM-powered applications. Real-time attack simulations on apps built with Python, Node.js, and PHP, showing how even small coding mistakes get exploited.
